Javascript Development Course

Developer Circles Jakarta from Facebook (Batch 14)

Session 1

Pengenalan HTML dan CSS

Mengenal konsep Markup Language, mengenal HTML dan mempelajari cara layouting HTML sederhana, serta mempercantik desain dengan CSS.

Session 2

Pengenalan JavaScript dan DOM

Membuat tampilan website yang interaktif dengan JavaScript dan DOM serta melatih logika pemrograman menggunakan JavaScript.

Session 3

JavaScript Basic Logics

Memperdalam logika pemrograman menggunakan JavaScript untuk mendukung perancangan dan pembuatan aplikasi yang lebih kompleks.

Session 4

JavaScript Basic Tools

Pengenalan berbagai tools JavaScript untuk menguatkan kemampuanmu dalam membuat website.
